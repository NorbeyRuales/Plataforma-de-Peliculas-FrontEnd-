Pages/Account

  • Loads the authenticated user's profile, lets them edit name/last name/age/email, and update their password in the same screen.
  • Adds guardrails (min/max age, email format) and basic A11Y patterns: focus management, aria-* error wiring, and role="dialog" for the delete modal.
  • WCAG notes:
    • 3.3.1/3.3.3: Error identification & hints (inline errors + hints).
    • 2.4.3: Focus sent to first field when entering edit mode.
    • 3.2.2: No unexpected context changes on input; actions require explicit user intent.
    • 2.1.1/2.1.2/2.4.7: Modal con focus-trap + Escape + foco inicial y restauración.
